# Online Trip Management System (PHP + MySQL + Bootstrap)

## Features
- User registration and login
- Trip creation and listing
- Trip booking with slot deduction
- My bookings page
- Clean Bootstrap UI

## Tech Stack
- Backend: PHP (PDO)
- Database: MySQL
- Frontend: Bootstrap 5

## Setup Steps
1. Put this folder in your web server root (for example `htdocs/online_trip_managemet_system` in XAMPP).
2. Create database/tables by importing `database.sql` in phpMyAdmin.
3. Open `config/db.php` and update DB username/password if needed.
4. Start Apache and MySQL.
5. Open: `http://localhost/online_trip_managemet_system/index.php`
